Problem-



write a program to find whether the given integer is odd or even.
Print "even" without quotes if input integer is even or "odd" if its odd.

INPUT-

An integer T denoting number of test cases,in next line
Two space separated integers a and b.

Constraints-
-10005<=a,b<=10005

Sample INPUT-

5
1
2
3
4
5

Sample Output-
odd
even
odd
even
odd
